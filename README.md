# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/event-portfolio.git
```

---

## Go to Project

```bash
cd event-portfolio
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

## Build Production

```bash
npm run build
```

---

## Start Production

```bash
npm start
```
